# Сайт электронной библиотеки Максима.

[Этот](https://vitaliusius.github.io/Html_03/pages/index1.html) отображает карточки книг Максима по 20шт на странице.

## Запуск сайта

Для запуска файла на ПК необходимо скачать [этот](https://github.com/Vitaliusius/Html_03/tree/main) репозиторий.

Для отображения карточек книг и корректной работы сайта необходима база данных книг, путь к которой и название файла необходимо присвоить переменной  в файле `.env` как в примере ниже:
```
PATH_TO_LIBRARY = 'library/meta_data.json'
```

Сервер запускается командой:

```
python render_website.py
```

При удачном запуске видим в консоли следующее:
```
[I 260416 15:16:32 server:331] Serving on http://127.0.0.1:5500
[I 260416 15:16:32 handlers:62] Start watching changes
[I 260416 15:16:32 handlers:64] Start detecting changes
```
Далее в браузере вводим адрес `http://127.0.0.1:5500/pages/index1.html`

Так же сайт работает без сервера. Чтобы запустить сайт, нужно запустить файл `index1.html` в папке `pages/`

## Окружение

Python3.14.2 уже должен быть установлен. Для изоляции проекта рекомендуется использовать virtualemv/venv. Для установки зависимостей:

```
pip install -r requirements.txt
```
