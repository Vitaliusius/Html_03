# Сайт библиотеки Максима.

Сайт отображает карточки книг Максима по 20шт на странице.

## Запуск сайта

Для отображения карточек книг и корректной работы сайта необходима база данных книг, которая хранится в корневом каталоге в файле `meta_data.json`

Сервер запускается командой:

```
python render_website.py
```

При удачном запуске видим в консоли следующее:
```
[I 260416 15:16:32 server:331] Serving on http://127.0.0.1:5500
[I 260416 15:16:32 handlers:62] Start watching changes
[I 260416 15:16:32 handlers:64] Start detecting changes
```
Далее в браузере вводим адрес `http://127.0.0.1:5500/pages/index1.html`

## Окружение

Python3.14.2 уже должен быть установлен. Для изоляции проекта рекомендуется использовать virtualemv/venv. Для установки зависимостей:

```
pip install -r requirements.txt
```
